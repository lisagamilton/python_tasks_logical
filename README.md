## 📊 Python for Data Analysis

This repository contains Python-based workflows and notebooks focused on data analysis, transformation, and exploration.

Most of the work here reflects practical data tasks: cleaning messy data, transforming it into usable formats, and exploring it to understand what’s going on.

---

## 🔧 What’s inside

- **Data cleaning & transformation**  
  Working with datasets using Pandas and NumPy: filtering, grouping, reshaping, and preparing data for analysis  

- **Exploratory data analysis (EDA)**  
  Looking for patterns, trends, and anything unusual in the data  

- **Data visualization**  
  Turning data into clear visual insights (mainly with Matplotlib)  

- **Working with databases**  
  Connecting to external data sources and running queries  

- **Data parsing**  
  Extracting and structuring data from different formats  

---

## 📁 Repository Structure

- `pandas_*.ipynb`

- `python_logic_practice.ipynb`
